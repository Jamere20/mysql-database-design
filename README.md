# Database Design and Management

This project demonstrates database creation, table relationships, and backend configuration for dynamic web applications using MySQL and phpMyAdmin.

## Topics Covered
- Database creation
- Relational table design
- Table schemas
- Primary and foreign keys
- Student registration database systems
- phpMyAdmin configuration
- Backend data structure planning

## Technologies Used
- MySQL
- phpMyAdmin
- SQL
- PHP

## Skills Demonstrated
- Database architecture
- Relational database management
- Backend configuration
- SQL concepts
- Data organization
- Dynamic website database integration

## Purpose
This project was created as part of Programming for the Web coursework to demonstrate backend database design and management concepts for dynamic web applications.
